# Percona ClusterSync for MongoDB Documentation Code of Conduct

All Percona Products follow the [Percona Community Code of Conduct :octicons-link-external-16:](https://percona.community/contribute/coc/).

If you experience any unacceptable behavior, let us know as soon as possible by emailing to <community-team@percona.com>. We’ll acknowledge your message promptly and respond within 48 hours.
