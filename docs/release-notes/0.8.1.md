### Bug

- [PCSM-305](https://perconadev.atlassian.net/browse/PCSM-305): We have fixed an issue where heavy replication workloads caused MongoDB nodes to crash due to the following error:

`Plan executor error during update :: caused by :: BufBuilder attempted to grow() to 131072004 bytes, past the 125MB limit.`

This crash led to write contention, timeouts, and stalled migrations, especially when moving large collections from Atlas.
